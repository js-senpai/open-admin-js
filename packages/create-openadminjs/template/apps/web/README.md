# OpenAdminJS Web Layer

This folder is part of the generated application, not the library marketing site.

Use it for:

- public frontend pages powered by OpenAdminJS API/resource metadata;
- custom pages for entities created in the admin;
- SEO metadata, canonical URLs, robots and sitemap generation;
- field-level public rendering where you choose which resource fields appear on each page.

The marketing landing/docs site for GitHub Pages lives in `/site`.
