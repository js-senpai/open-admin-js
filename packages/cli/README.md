# openadminjs CLI

Official CLI for OpenAdminJS.

## Quick start

```bash
npx openadminjs create my-app
cd my-app
pnpm db:migrate
pnpm db:seed
pnpm dev
```

## Commands

- `openadminjs create [projectName]`
- `openadminjs dev`
- `openadminjs build`
- `openadminjs start`
- `openadminjs db migrate|seed|studio`
- `openadminjs generate resource <ModelName>`
- `openadminjs generate plugin <pluginId>`
- `openadminjs doctor`
- `openadminjs security check`
